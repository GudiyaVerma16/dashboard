## React Dashboard (Vite + TypeScript)

This folder contains a modern analytics-style dashboard UI built with **Vite + React (TypeScript)** and styled using **Tailwind CSS v3**.

### Run locally

- **Install dependencies**

```bash
npm install
```

- **Start dev server**

```bash
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`) in your browser.

- **Build for production**

```bash
npm run build
```

### Features

- Modern, responsive dashboard UI
- Dark/Light theme toggle
- Interactive charts and data visualizations
- Reusable components with TypeScript
- Smooth animations with Framer Motion
